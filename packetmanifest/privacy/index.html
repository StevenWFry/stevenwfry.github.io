Name
PacketManifest — privacy policy
Last updated: March 11, 2026
Description

PacketManifest does not collect, store, or share your personal data.

Data Storage

All network connection records — including hostnames, IP addresses, ports, and location data — are stored locally in a database on your device. This data is never transmitted to any server controlled by the developer.

GeoIP Lookups

To display the geographic location of remote servers, PacketManifest sends remote IP addresses (not your IP address) to a private GeoIP server at geoip.swf.wtf. These lookups are not logged or retained.

Analytics

PacketManifest contains no analytics SDKs, advertising frameworks, or crash reporting tools.

VPN Usage

PacketManifest uses iOS's Network Extension framework to create a local VPN tunnel on your device. This tunnel is used solely to observe outbound connections. No traffic is routed to an external VPN server.
