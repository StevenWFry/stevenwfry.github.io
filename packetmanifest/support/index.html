Common Questions
Monitor button doesn't work
PacketManifest requires a VPN configuration to be installed. Tap Monitor and accept the VPN prompt when it appears. If the prompt doesn't appear, go to Settings → VPN and check if a PacketManifest configuration exists.
No connections showing
Make sure monitoring is active (green dot in the toolbar). Try opening Safari or another app to generate traffic.
Map shows wrong home location
The home pin is based on your public IP address and may not be precise. It is used only as the origin point for map arcs.
Battery or data usage
The local VPN tunnel adds minimal overhead. If you're concerned, stop monitoring when not in use by tapping Stop.
